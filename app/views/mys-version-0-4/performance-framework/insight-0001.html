Insight

The data from copy and paste analysis of fields showed that users are frequently copying and pasting into text boxes.

This suggests that they are using the headings (or something else) into these fields but was not conclusive.

To get more conclusive evidence we performed text analysis of the content of the completed fields.

This analysis showed conclusively that users were copying and pasting headers into fields.
